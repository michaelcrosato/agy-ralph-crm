# Task 0168: Forecast Adjustments & Manager Target Overrides API - Brief

## Objective
Establish a Forecast Adjustment & Manager Target Overrides Engine for the CRM Core system under Phase 6. This enables sales managers and directors to submit adjustments/overrides to calculated opportunity forecasting and representative quotas for active planning periods, and retrieve aggregated forecast summaries with these adjustments dynamically applied.

## Core Value
- **Manager Overrides**: Allows sales managers to apply their judgement to representative forecast numbers and period quotas, ensuring more accurate planning.
- **Dynamic Calculation**: Integrates manager adjustments directly into the forecasting and leaderboard aggregation systems.
- **Tenant Context and RLS Protection**: Fully runs under active tenant row-level security isolation, ensuring adjustments and overridden values are completely private and secure.
