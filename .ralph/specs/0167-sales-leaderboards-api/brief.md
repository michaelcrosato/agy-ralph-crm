# Task 0167: Sales Leaderboards & Quota Attainment API - Brief

## Objective
Establish a high-value Sales Leaderboard & Quota Attainment Analytics Engine for the CRM Core system under Phase 6. This enables sales leaders, managers, and representatives to dynamically view and aggregate closed-won opportunity totals by representative for any specified month or quarter, compare these figures against active sales quotas, compute quota attainment percentages, rank individual performance, and retrieve tenant-isolated leaderboard analytics.

## Core Value
- **Real-Time Performance Tracking**: Provides direct visibility into closed-won opportunity sales performance compared against periods quotas.
- **Dynamic Periods Support**: Supports tracking sales performance and quotas by both months (e.g. `2026-05`) and quarters (e.g. `2026-Q2`).
- **Tenant Context and RLS Protection**: Fully runs under active tenant row-level security isolation, ensuring leaderboard data, users, and quota targets are completely private and secure.
